## Ejecutar en desarrollo 
````
1. Clonar el repositorio
````
2. renombrar el archivo .env-templete a .env
``````
3. ejecutar
``````
yarn install 
``````
4. Tener Nest CLI instalado

``````
npm i -g @nestjs/cli
``````

5. levantar base de datos
``````
docker-compose up -d
``````
6. Levantar la aplicacion
``````
yarn start:dev
``````
